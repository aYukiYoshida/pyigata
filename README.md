# PyIgata

## 諸情報

### 環境設定

* 仮想環境準備
    ```
    % virtualenv venv
    % source ./venv/bin/activate
    % deactivate  # 仮想環境終了
    ```
If you want to learn more about `setup.py` files, check out [this repository](https://github.com/kennethreitz/setup.py).

* build
    ```
    % ./setup.sh build
    ```
* install
    ```
    % ./setup.sh install
    ```
* unit test
    ```
    % ./setup.sh test
    ```
* clean
    ```
    % ./setup.sh clean
    ```
* check
    ```
    % ./setup.sh check
    ```
* generate package
    ```
    % ./setup.sh sdist
    ```
    The package is generated in dist/
